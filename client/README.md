# Admission Management & CRM System

A complete MERN stack application for managing college admissions with quota control, seat allocation, and dashboards.

## Features

- **Institution Management**: Create institutions, campuses, departments
- **Program Setup**: Configure programs with quotas and intake limits
- **Applicant Management**: Register applicants with 15 essential fields
- **Seat Allocation**: Allocate seats with quota validation
- **Admission Workflow**: Generate admission numbers and confirm admissions
- **Real-time Updates**: Socket.io for live seat counter updates
- **Dashboard**: View admission statistics and analytics
- **Role-based Access**: Admin, Admission Officer, Management views

## Tech Stack

- **Frontend**: React 18, Redux Toolkit, Ant Design
- **Backend**: Node.js, Express, MongoDB
- **Database**: MongoDB
- **Real-time**: Socket.io
- **Authentication**: JWT
- **API Client**: Axios
